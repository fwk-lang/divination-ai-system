DEPLOYMENT.md  # 命理AI系統 - 部署指南

## 快速開始

### 本地運行
```bash
# 克隆專案
git clone https://github.com/fwk-lang/divination-ai-system.git
cd divination-ai-system

# 安裝依賴
pip install -r requirements.txt

# 啟動應用
streamlit run app.py
```

### Streamlit Cloud部署
1. Fork此專案
2. 連結Streamlit Cloud
3. 選擇app.py作為主程式
4. 一鍵部署
